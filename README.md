# Lista de Exercícios - Métodos de Array (JavaScript ES6+)

Este projeto contém exercícios práticos em JavaScript focados no uso de métodos de array com Arrow Functions.

---

## Objetivo

Praticar e consolidar o uso de métodos fundamentais de manipulação de arrays:

- `map()`
- `filter()`
- `find()`
- `reduce()`
- `some()`
- `every()`

Além disso, reforçar:

- Arrow Functions
- Lógica funcional
- Transformação de dados
- Validação de listas

---

## Importância para o desenvolvimento

Esses métodos são essenciais no JavaScript moderno e são amplamente usados em:

- React (renderização de listas e manipulação de estado)
- Consumo de APIs
- Processamento de dados
- Lógica de aplicações web

---

## Estrutura dos Exercícios

Os exercícios podem ser organizados em um único arquivo:


exercicios.js


---

## Conteúdo dos Exercícios

### 1. Filtrar positivos
Uso do `filter()` para selecionar apenas números positivos.

### 2. Nomes em maiúsculo
Uso do `map()` para transformar strings.

### 3. Busca de usuário
Uso do `find()` para localizar um objeto específico.

### 4. Cálculo de imposto
Uso do `map()` para alterar valores num array.

### 5. Contagem de caracteres
Transformação de strings em tamanhos numéricos.

### 6. Filtrar intervalo de valores
Uso do `filter()` com múltiplas condições.

### 7. Soma de carrinho
Uso do `reduce()` para somar valores.

### 8. Verificar existência
Uso do `some()` para validar condição em arrays.

### 9. Validação de lista
Uso do `every()` para verificar se todos os itens atendem a uma condição.

### 10. Objeto curto
Criação de objeto usando Arrow Function com retorno implícito.
